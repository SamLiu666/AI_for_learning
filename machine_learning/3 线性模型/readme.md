4、线性判别分析 -- LDA

**思想：** 给定训练样本集合，设法将样例投影到一条直线上，使得同类样例的投影点尽可能接近，异样投影点尽可能远离

6、类别不平衡问题

解决办法：欠采样(去除一些样本，使得正反数目接近)、过采样(增加一些样本，使得正反数目接近)、再缩放、权重设置、损失函数设置
