3、优化方法

3.1 梯度下降：挑选一个初始值w0，重复迭代次数，沿梯度方向增加损失函数值

学习率：，太小--学习效果有限，计算消耗太大，太大--振荡，学习不到优化

3.2 小批量随机梯度下降：随机采样b个样本来近似损失，不能太小，不能太大

学习率，批量 -- **超参数**
